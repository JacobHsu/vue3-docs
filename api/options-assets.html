# 选项/资源

# directives

  • 类型：Object

  • 详细：

    声明组件实例中可用指令的哈希表。

  • 用法：

    const app = createApp({})

app.component('focused-input', {
  directives: {
    focus: {
      mounted(el) {
        el.focus()
      }
    }
  },
  template: `<input v-focus>`
})

# components

  • 类型：Object

  • 详细：

    声明组件实例中可用组件的哈希表。

  • 用法：

    const Foo = {
  template: `<div>Foo</div>`
}

const app = createApp({
  components: {
    Foo
  },
  template: `<Foo />`
})
